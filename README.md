# CyberScan Pro

أداة احترافية لفحص أمن المواقع (SSL & Security Headers). 
هذا المشروع مصمم ليكون أساساً قوياً للمطورين وأصحاب المواقع.

## المتطلبات:
- Python 3.x
- المكتبات: fastapi, uvicorn, requests

## طريقة التشغيل:
1. افتح التيرمينال في مجلد المشروع.
2. قم بتثبيت المكتبات:
   `pip install -r requirements.txt`
3. شغل السيرفر:
   `python -m uvicorn main:app --reload`
4. افتح المتصفح على الرابط: `http://127.0.0.1:8000`
